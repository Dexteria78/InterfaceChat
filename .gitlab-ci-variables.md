# Variables nécessaires dans GitLab CI/CD Settings

## Azure
- AZURE_CLIENT_ID
- AZURE_CLIENT_SECRET
- AZURE_TENANT_ID

## Azure Container Registry
- ACR_LOGIN_SERVER
- ACR_USERNAME
- ACR_PASSWORD

## Kubernetes
- KUBE_CONFIG (base64 encoded kubeconfig)

## Optionnel
- SLACK_WEBHOOK (notifications)
- TRELLO_API_KEY (intégration Trello)
- TRELLO_TOKEN
